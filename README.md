 Project Introduction

This is a Real-Time Weather Application built using HTML, CSS, and JavaScript, integrated with the OpenWeather API to fetch live weather data of any city in the world.
The user can simply enter a city name and instantly view detailed weather parameters such as temperature, humidity, wind speed, pressure, and current weather conditions.

This project demonstrates how to work with third-party APIs, handle JSON data, and build a clean, responsive UI using modern web technologies.
